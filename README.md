# Music Emotion Recognition(MER)
Questo progetto esplora il riconoscimento delle emozioni musicali utilizzando diverse architetture di Deep Learning (CNN, VGG16) e diverse tecniche di estrazione (Mel Spectrograms, MFCC).
Il repository è organizzato come segue:
- confusion_matrix contiene le	matrici di confusione per CNN standard, VGG16 e CNN con MFCC.
- from_youtube contiene spettrogrammi e predizioni ottenute dall'analisi di 3 brani estratti da YouTube.
- mel_spectrogram contiene le visualizzazioni degli spettrogrammi delle diverse emozioni e confronto tra MFCC e Mel Spectrogram.

Il file Article_MER.pdf contiene l'articolo scientifico completo che documenta il progetto.
Il file .ipynb è stato salvato con l'output delle celle già visibile e contenente ulteriori approfondimenti: Waveform ed analisi degli errori.
